DocuAgent Chat Widget — Beágyazás

Másold be az alábbi kódot a weboldalad </body> zárótag elé. A widget egy lebegő chat bubble-ként jelenik meg az oldal jobb alsó sarkában.

📌 Iframe embed (ajánlott)

Ez a módszer egy iframe-et helyez el az oldalon. Cseréld ki a placeholdereket a saját értékeiddel.

<!-- DocuAgent Chat Widget -->
<style>
  #da-iframe {
    position: fixed; bottom: 0; right: 0;
    width: 420px; height: 620px;
    border: none; z-index: 9999;
    background: transparent;
  }
  @media (max-width: 480px) {
    #da-iframe { width: 100vw; height: 100vh; }
  }
</style>

<iframe
  id="da-iframe"
  src="https://YOUR_DOMAIN/widget.html?tenant=YOUR_SLUG&color=%231a56db&api_url=https://YOUR_API"
  allow="microphone"
  title="DocuAgent Chat"
></iframe>

🔑 Paraméterek

tenant      = "demo"       # a tenant slug
color       = "%231a56db"  # primary szín (URL-encode a #-t → %23)
api_url     = "https://…"  # a backend URL (opcionális, ha same-origin)
💡 Tipp: A testreszabható értékeket (szín, üdvözlőüzenet) a DocuAgent admin felületen az Integrációk → Widget szekcióban állíthatod be, és ezek automatikusan betöltődnek a widgetbe.

▶ Élő példa (localhost)

Az alábbi widget ezt az oldalt demonstrálja, demo tenant-tal.
